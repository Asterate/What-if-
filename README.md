# "what if?" project

Idea is to make a simple app inspired by a book "what if?". It would display at chosen frequency a silly question and when you tap on it you see an answer like Anki cards. I am planning to make for this version just set in stone questions which will not be updated just for demo purposes. I have added timer for 6 hours after every question because it is meant to be not scrolled through straight away to make it more interesting for longer time usage.

## Project structure

The project will have 3 files:
- main file for app antry point
- Home page file for the main view
- questions file for the content

## Usage
In terminal put flutter run and I recommend choosing Chrome for web usage.

For Android phone
1. Put your phone in developer mode and turn on USB debugging
2. Have usb-c cable that can transfer data
3. Connect it to the computer and mobile
4. In codebase terminal type in flutter devices to see if phone is connected. If it fails turn debugging off and on and replug the cable.
5. If device is detected use its id which is in second column and type in terminal: flutter run -d "device id"
6. It will take a while first time and cannot let phone turn off.
